# 剧本杀小程序

一个基于 uniapp + Vue2 的微信小程序，专为剧本杀爱好者打造的预约平台。

## 功能特点

### 🎭 核心功能
- **剧本浏览**：精选剧本库，支持分类筛选和排序
- **场次预约**：实时查看可用场次，快速预约
- **订单管理**：查看预约记录，支持修改和取消
- **个人中心**：用户信息管理，会员特权

### 🎨 界面设计
- **现代化UI**：渐变配色，圆角卡片设计
- **响应式布局**：适配不同屏幕尺寸
- **流畅动画**：提升用户体验
- **直观导航**：底部Tab栏，便于操作

### 📱 页面结构

#### 首页 (pages/index/index.vue)
- 轮播横幅展示
- 功能快捷入口
- 热门剧本推荐
- 今日场次预览
- 店铺基本信息

#### 剧本列表 (pages/scripts/scripts.vue)
- 分类筛选（悬疑、古风、现代等）
- 多维度排序（热度、价格、难度、时长）
- 剧本详情预览
- 在线预约入口

#### 剧本详情 (pages/script-detail/script-detail.vue)
- 高清封面展示
- 详细剧情介绍
- 角色信息展示
- 用户评价展示
- 今日场次选择
- 一键预约功能

#### 预约页面 (pages/reservation/reservation.vue)
- 剧本选择向导
- 日期时间选择
- 场次状态查看
- 联系信息填写
- 价格明细展示
- 预约须知说明

#### 我的预约 (pages/orders/orders.vue)
- 订单状态筛选
- 预约详情查看
- 倒计时提醒
- 订单操作管理
- 客服联系方式

#### 个人中心 (pages/profile/profile.vue)
- 用户信息展示
- 游戏统计数据
- VIP会员特权
- 功能菜单导航
- 设置和帮助

## 技术特点

### 🛠 技术栈
- **框架**：uniapp
- **前端**：Vue2
- **样式**：CSS3，支持渐变和动画
- **平台**：微信小程序

### 🎯 设计理念
- **用户友好**：简洁直观的操作流程
- **性能优化**：合理的数据结构和加载策略
- **扩展性强**：模块化组件设计
- **维护便利**：清晰的代码结构

### 📊 数据模拟
项目包含丰富的模拟数据：
- 剧本信息：名称、封面、标签、难度、价格等
- 场次安排：时间、人数、状态等
- 用户数据：个人信息、游戏记录、会员状态等
- 订单信息：预约详情、状态跟踪等

## 开发指南

### 📋 环境要求
- Node.js 12+
- HBuilderX 或 其他支持 uniapp 的开发工具
- 微信开发者工具

### 🚀 快速开始
1. 导入项目到开发工具
2. 配置微信小程序 appid
3. 运行到微信开发者工具
4. 开始开发调试

### 🔧 配置说明
- `manifest.json`：应用配置和权限设置
- `pages.json`：页面路由和导航配置
- `App.vue`：全局样式和生命周期
