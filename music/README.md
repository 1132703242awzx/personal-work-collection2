# 云音乐 - 精美的Android音乐播放器

一个现代化的Android音乐播放器应用，采用Material Design 3设计，提供流畅的用户体验。

## 🎵 功能特点

### 核心功能
- **音乐播放** - 支持播放、暂停、上一首、下一首
- **播放模式** - 支持顺序播放、随机播放、单曲循环、列表循环
- **进度控制** - 可拖拽进度条调整播放位置
- **音量控制** - 系统音量控制集成

### 界面设计
- **精美UI** - 采用Material Design 3设计语言
- **深色主题** - 护眼的深色主题设计
- **渐变背景** - 美观的渐变色背景
- **流畅动画** - 丝滑的页面切换和交互动画

### 主要页面
- **首页** - 推荐歌单、最近播放、热门歌曲
- **搜索** - 搜索歌曲、歌手、专辑
- **收藏** - 管理收藏的歌曲
- **我的** - 个人信息、播放统计、设置

### 播放器功能
- **全屏播放器** - 大图封面、完整控制面板
- **迷你播放器** - 底部悬浮播放条
- **歌词显示** - 同步歌词显示（预留功能）
- **播放队列** - 当前播放队列管理

## 🏗️ 技术架构

### 技术栈
- **Kotlin** - 100% Kotlin开发
- **Jetpack Compose** - 现代化的UI工具包
- **Material Design 3** - 最新的设计系统
- **Architecture Components** - ViewModel、LiveData、Navigation
- **Coroutines** - 异步编程
- **StateFlow** - 状态管理

### 项目结构
```
app/
├── src/main/java/com/example/music/
│   ├── data/           # 数据模型
│   ├── ui/
│   │   ├── navigation/ # 导航组件
│   │   ├── player/     # 播放器组件
│   │   ├── screen/     # 页面组件
│   │   └── theme/      # 主题配置
│   ├── viewmodel/      # 视图模型
│   └── MainActivity.kt # 主活动
└── src/main/res/       # 资源文件
```

### 依赖库
- **Compose BOM** - Compose组件管理

- **Coil** - 图片加载
- **Lottie** - 动画效果

## 🚀 如何运行

### 环境要求
- Android Studio Hedgehog或更高版本
- Kotlin 1.9.0或更高版本
- Android SDK 24或更高版本
- JDK 11或更高版本

### 运行步骤
1. 克隆项目到本地
2. 使用Android Studio打开项目
3. 等待Gradle同步完成
4. 连接Android设备或启动模拟器
5. 点击运行按钮

### 编译命令
```bash
./gradlew assembleDebug      # 编译Debug版本
./gradlew assembleRelease    # 编译Release版本
```
